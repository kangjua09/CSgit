# 🍜 매일매일 고민되는 점심메뉴 추천 및 관리 시스템

## 1. 프로젝트 소개
매일 반복되는 점심 메뉴 선정 과정에서 오는 결정 장애를 해결하기 위해, 사용자의 기분, 날씨, 예산 등의 **다중 조건을 필터링**하여 **최적의 메뉴를 추천**하고 **데이터를 영구적으로 관리**하는 시스템입니다.

## 2. 주요 기능
* **메뉴 추천:** 카테고리, 가격대, 맵기, 국물 유무 등 사용자가 선택한 **모든 조건(AND)**을 만족하는 식당 중 하나를 무작위로 추천합니다.
* **데이터 관리:** 새로운 맛집 정보를 입력받아 데이터베이스에 추가하고 JSON 파일에 저장합니다.
* **안정성 확보:** 파일이 없을 경우 기본 리스트를 생성하며, 사용자의 잘못된 입력(타입 오류)에도 프로그램이 종료되지 않도록 예외 처리(try-except)를 적용했습니다.

## 3. 사용된 기술
* **Python:** 프로젝트의 주 언어
* **json:** 데이터(Dictionary, List)를 파일에 구조화하여 저장하고 불러오는 데 사용
* **os:** 데이터 파일의 존재 여부 확인 및 경로 관리
* **random:** 필터링된 후보군 내에서 최종 메뉴를 공정하게 무작위 선정

## 4. 실행 방법
프로젝트 파일을 다운로드 후, 터미널(Terminal) 또는 명령 프롬프트(Command Prompt)에서 다음 코드를 입력하여 실행합니다.

```bash
python lunch_selector.py
